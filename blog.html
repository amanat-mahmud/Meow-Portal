Difference between var,let & const

Var was only way of declaring variable. Later in ES6 let and const was added.Var had hoisting problem. But let and const didn't as a result they became famous.Var can be reassigned and reinitialized. Let and const are block scoped variable. Let can be reassigned not reinitialized. Const can't be reassigned or reinitialized.

Difference between function and Arrow function.

Arrow function and normal function are sytantically different. Arrow function was introduced in ES6. Arrow function has no this property of it's own. When this is called from arrow function window is returned.Arrow function has implicit and explicit return.

Why we need to use template string?

When we use normal string we can't use any kind of expression inside them. Template string solves this problem. By using template string writting multiline string has become easies. Template string makes our string dynamic and makes our life easies. Because of all these reasons we use template string.

Difference between map(), forEach(), filter() and find() function

map() is used when we want to modify our array and store it on another one it returns variables one by one.forEach() is also used to modify array values but it does not return anything mainly used to loop through an array. filter() is used when we want an array where that array elements satisfy a particular condition. find() works just like the filter() but instead returns the first element of the array that satisfies the condition.
